# Cizen

Build highly concurrent, monitorable, and extensible applications
with a collection of automata.

"Cizen" means "Nature" in Japanese.  
/'sizen/ 自然

## Installation

First, add Cizen to your list of dependencies in `mix.exs`:

```elixir
defp deps do
  [
    {:cizen, "~> 0.1.0"}
  ]
end
```

Then, run `mix deps.get`.
